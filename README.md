# Success Dynamics Accountability Chart (Streamlit) — v4

**New:** Monthly Tracking (plan vs actuals, COGS, fixed people costs, other overheads) with projections.

## Quickstart
```bash
pip install -r requirements.txt
streamlit run app.py
```

## GitHub Upload (drag & drop)
1. Download / unzip this folder locally.
2. On GitHub, **Add file → Upload files**.
3. Drag all files **including the `data` folder**.
   - `.gitkeep` files ensure `data/profiles` and `data/logos` are created.
4. Commit.

## Manual Git (PowerShell) — no script required
```powershell
# 1) Open PowerShell in the unzipped folder (Shift+Right‑click → 'Open PowerShell here')
# 2) Then run:
git init
git add .
git commit -m "Initial commit: Success Dynamics Accountability Chart v4"
git branch -M main
git remote add origin https://github.com/USERNAME/REPO.git
git push -u origin main
```
If you see an authentication prompt, sign in with your GitHub token or GitHub Desktop.
